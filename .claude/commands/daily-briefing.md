# Daily Briefing

You are the AI Employee. Generate a comprehensive daily briefing for the human.

## Instructions

1. **Read** `AI_Employee_Vault/Dashboard.md` for current system status
2. **Read** `AI_Employee_Vault/Business_Goals.md` for targets and metrics
3. **Scan** `AI_Employee_Vault/Done/` for tasks completed today
4. **Scan** `AI_Employee_Vault/Needs_Action/` for pending items
5. **Scan** `AI_Employee_Vault/Pending_Approval/` for items awaiting human decision
6. **Read** today's log file from `AI_Employee_Vault/Logs/YYYY-MM-DD.json` if it exists

## Generate a Briefing File

Write the briefing to `AI_Employee_Vault/Briefings/YYYY-MM-DD_Daily_Briefing.md` using this format:

```markdown
---
generated: <ISO timestamp>
period: <date range>
---

# Daily AI Employee Briefing — <Date>

## Executive Summary
<2-3 sentence summary of the day>

## Completed Tasks
<list tasks from /Done>

## Pending Actions
<list items in /Needs_Action>

## Awaiting Your Approval
<list items in /Pending_Approval>

## Alerts & Recommendations
<any issues, suggestions, or flags>

---
*Generated by AI Employee v0.1*
```

3. **Update** `AI_Employee_Vault/Dashboard.md` with the latest status
4. **Report** the briefing contents to the user in the terminal

## Output

Print a concise summary and tell the user where the full briefing was saved.
