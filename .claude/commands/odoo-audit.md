# Odoo Accounting Audit

You are the AI Employee. Your job is to run an accounting audit using Odoo and generate a financial report.

## Prerequisites Check

First verify Odoo is configured:
1. Check `mcp_servers/odoo_mcp/config.json` exists (copy from `config.example.json`)
2. Check Odoo is running at `http://localhost:8069`
3. Check `odoo` MCP server is in `.claude/settings.json`

If not configured, guide the user:
```
See mcp_servers/odoo_mcp/README.md for setup instructions.
Quick start with Docker:
  docker run -p 8069:8069 -e POSTGRES_USER=odoo -e POSTGRES_PASSWORD=odoo odoo:17
```

## When Odoo IS Configured

Use the Odoo MCP server tools:

### 1. Revenue Summary
```
Tool: get_financial_summary
Parameters: { "period": "this_month" }
```

### 2. Outstanding Invoices
```
Tool: list_invoices
Parameters: { "state": "posted", "payment_state": "not_paid" }
```

### 3. New Customers This Week
```
Tool: list_customers
Parameters: { "limit": 10, "order": "create_date desc" }
```

### 4. Product Inventory
```
Tool: list_products
Parameters: { "limit": 20 }
```

## When Odoo is NOT Configured (Fallback)

Run the manual audit using vault files:
1. Read `AI_Employee_Vault/Accounting/` for existing records
2. Read `AI_Employee_Vault/Invoices/` for invoice history
3. Read `AI_Employee_Vault/Business_Goals.md` for targets
4. Calculate MTD revenue vs target

- Gap: $X,XXX (XX% of target)

*Generated by AI Employee Gold Tier — Odoo MCP*
